# wp.org listing assets

Place wordpress.org listing assets here (the `_release-free.yml` workflow uploads this dir as
`ASSETS_DIR`). Use the zoomed-out, colorful demo-theme capture style (see the storefront roadmap).

- `icon-256x256.png` (and optional `icon-128x128.png`)
- `banner-772x250.png` (and optional `banner-1544x500.png`)
- `screenshot-1.png`, `screenshot-2.png`, … (match the order in readme.txt)
